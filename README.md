## Overview

Eva Devs Management System is a complete solution that manages all academy operations. Through this system, you can:

* Add and manage students
* Add and manage courses
* Enroll students in courses
* Track daily attendance
* Manage the fee system
* View detailed reports

## Core Features

## Student Management

* You can add students
* Search students (by Name, Father Name)
* When a student is deleted, their data is moved to the **Previous Section**

---

## Course Management

* You can add courses
* You can check which courses are being offered in your academy
* Course-wise student data

---

## Enrollment System

* You can enroll students in courses
* Enrollment search is available
* Course-wise filter available**
*  Enrollment date filter available**

---

## Attendance System

* You can mark daily attendance
* Attendance filters:

  * Date-wise
  * Present only
  * Absent only
  * Leave only

---

## Fee Management
* You can add student fees
* Fee filters:
* 
  * Month-wise
  * Payment date-wise
  * Fee type

* Fee tracking:
  * Total Paid Fee
  * Total Unpaid Fee
  * Remaining Fee
  * Student Fee History

## Search & Filters

The system provides advanced search and filtering options:

* Student section:
  * Search by Name / Father Name
* Course section:
  * Search
  * Duration filter
  * Fee sorting (High → Low, Low → High)

* Enrollment:

  * Search
  * Course-wise filter
  * Enrollment date filter

* Attendance:

  * Search
  * Date-wise filter
  * Status filters (Present / Absent / Leave)

* Fee:

  * Month-wise filter
  * Payment date filter
  * Fee type filter

---

## Previous Section
If a student is deleted, their:
* Enrollment data
* Fee records
* Attendance
* Student record

are automatically moved to the **Previous Section**.
